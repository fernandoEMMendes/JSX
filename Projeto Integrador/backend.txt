Passos para criar uma tabela


1) criação do model (modelo) que será utilizado para fazer
as ligações com a tabela e criação do @@map (nome da tabela)

2) criação do services: (banco de dados)

	interface (tipificação dos dados)	
	class (conjuto de regras para certa função)
		consultas(para não ouver duplicidade, etc..)		
		ação que vai executar (Execute, Create, etc..
	mensagem de retorno ao front-end


3) criação do controller: (front-end)
	
	selecionar o midway (req e res) 
	class
	 	handle (manipular dados)
